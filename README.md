Run setup.sh to push shell/.bashrc and editors/.vimrc to $HOME, and to push ssh/authorized_keys and ssh/config to $HOME/.ssh/

```
[bl@bl-dt ~/dev/dotfiles] $
.
├── .gitignore
├── editors
│   └── .vimrc
├── setup.sh
├── shell
│   ├── .bashrc
│   └── .zshrc
└── ssh
    ├── authorized_keys
    └── config
```
